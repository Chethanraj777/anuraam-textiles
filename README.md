# AnuRaam Textiles

This is a simple online saree store built using HTML, CSS, and JavaScript. It was made as a practice project to simulate a basic e-commerce website.

## What this project does

- Shows a collection of sarees with images and prices  
- Lets users add items to a cart  
- Allows quantity increase and decrease  
- Calculates total price automatically  
- Shows product details in a popup view  
- Sends order details through WhatsApp  

## Products included

The site includes different types of sarees like:
- Kanchipuram Silk
- Banarasi Sarees
- Cotton Sarees
- Designer Wear Sarees

## How it works

Everything runs on the browser. The cart data is stored locally, so it stays even after refresh. Orders are sent through WhatsApp instead of a payment gateway.

## Tech used

- HTML
- CSS
- JavaScript

## Note

This is not a full production store. It’s a front-end project to practice web development and UI design.
